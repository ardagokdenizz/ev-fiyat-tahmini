# 🏠 Ev Fiyat Tahmini Projesi (House Price Prediction)

## 📌 Proje Amacı
Bu proje, makine öğrenmesi kullanarak evlerin özelliklerine göre satış fiyatlarını tahmin etmeyi amaçlamaktadır.

---

## 📊 Kullanılan Veri Seti
Kaggle - House Prices: Advanced Regression Techniques  
https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques

---

## 🧠 Problem Türü
Regresyon (Regression Problem)

Amaç: Sürekli bir değer olan ev fiyatını tahmin etmek.

---

## 🤖 Kullanılan Modeller

- Linear Regression
- Random Forest Regressor

---

## 📈 Model Performansı

- Linear Regression MAE: ~23000
- Random Forest MAE: ~18100

---

## 🏆 Sonuç

Random Forest modeli, Linear Regression modeline göre daha düşük hata üretmiş ve daha başarılı performans göstermiştir.  

Bu durum, veri içindeki doğrusal olmayan ilişkilerin Random Forest tarafından daha iyi öğrenildiğini göstermektedir.

---

## 📊 Kullanılan Değerlendirme Metrikleri

- MAE (Mean Absolute Error)
- R² Score (opsiyonel eklenebilir)
- RMSE (opsiyonel eklenebilir)
